# Unit 5.3 M4 Task 7 

## Instructions
- You are given a binary tree. Write a function that returns the binary tree's node values using an in-order traversal.

## Example:
Input: [2,None,3,4]
```
   2
    \
     3
    /
   4
   ```
Output: [2,4,3]

## Constraints
- [execution time limit] 4 seconds (py3)

- [input] tree.integer root

- [output] array.integer
